It showcases my projects, skills, and basic web development knowledge.

- 🎨 Styled using CSS
- ⚡ Interactive using JavaScript

- Subtraction (-)
- Multiplication (*)
- Division (/)

- Instant feedback (Correct / Try Again)
- Random number generation using JavaScript
